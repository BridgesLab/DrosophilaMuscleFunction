Experimental Design

This was for the second batch of knockdown crosses in which only homozygous drivers were used.

These data are the raw data saved in ../Data/Birth Data.csv for the births and ../Data/Pupae Inspection Data.csv for the pupae inspection. This script was most recently run on Fri Jun 27 08:55:39 2014.

These aggregaged and annotated data are written out to the file ../data/processed/Homozygous Crosses Pupae Data.csv.
